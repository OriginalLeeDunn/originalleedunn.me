Installation and Setup

Get started with the Digital Assistant by following these simple steps.

Prerequisites

  • Node.js 18+ or 20+
  • npm or yarn

Installation Steps

# Clone the repository
git clone https://github.com/yourusername/digital-assistant.git
cd digital-assistant
# Install dependencies
npm install
# Start the development server
npm run dev
# Build for production
npm run build
# Preview the production build
npm run preview
